# Discord-Bot

스프링 부트(Spring Boot) 기반의 **다시능 디스코드 봇** 프로젝트

## Tech Stack
* **Language:** Java 17
* **Framework:** Spring Boot 4.0.1
* **Library:** JDA (Java Discord API)
* **API:** 공공데이터포털 Open API (기상청 단기예보 / 에어코리아 대기오염정보 )


## 주요 기능
* **실시간 정보 조회:** 특정 명령어 입력 시 미세먼지 등 공공데이터 정보 제공
* **스케줄러 알림:** 설정된 시간에 맞춰 자동 알림(온도, 미세먼지)
